# Profiles Rating App

Welcome to the profiles rating App. This App is built in Vue 3.0 and uses Vuex for the state management.

If you ant to see the app runing, follow these steps:

- run `npm install` to setup the project
- run `npm run serve` to run the dev server in localhost:8080 and see the app runing.

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
